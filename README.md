# VMLY&R - Vue 3 Component UI Library Boilerplate

This is a front-end component ui library build in [Vue 3](https://vuejs.org/) + [Typescript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/) + [Tailwindcss](https://tailwindcss.com/)
 
## Components list

### Typography
1. Button

### Form elements
1. Form
2. Input
3. Textarea
4. Checkbox
5. Radio
6. Select

### Components
1. Accordion
2. Carousel
3. Modal
4. Pagination
5. Sliders
6. Tabs
7. Tooltip

## Installation
To install, run `yarn install` or `npm install` in the command line

## Development & Documentation
To run the development server, run `yarn dev` or `npm run dev`in the command line.
This is also how you can view the documentation.

## Production Build
To build for production, run `yarn build` or `npm run build` in the command line
